# Letterboxd Stats

Your Letterboxd, dissected & compared.

## Development

This project is built with:
- Next.js
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
